## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to play.

## How to Play

1. Click "Create New Game" to start
2. Share the game code or link with your friend
3. Both players can make moves in real-time
4. That's it! Simple and clean.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- chess.js (game logic)
- react-chessboard (board UI)
- Stockfish.js (chess engine AI)
- Framer Motion (animations)

## Design

- Unique fonts: Space Grotesk for headings, Inter for body
- Clean color palette with neutral tones and blue accents
- Simple, intuitive interface
- No dark mode or complex effects - just clean design
